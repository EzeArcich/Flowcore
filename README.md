# Flowcore CRM

Stop losing deals because you forgot to follow up.

Flowcore is a lightweight CRM built for freelancers, developers and small teams who need to manage companies, contacts, conversations and deals — without losing context.

---

## 🚀 What problem does it solve?

Most small teams don’t fail because they lack opportunities.

They fail because:
- conversations get lost
- follow-ups are forgotten
- deals lose momentum
- everything is scattered across tools

Flowcore solves that by centralizing your entire commercial flow in one place.

---

## ⚙️ What you can do with it

- Track companies and contacts
- Log every interaction (emails, messages, calls)
- Schedule follow-ups with due dates
- Manage quotations and deal status
- Operate your pipeline from a single dashboard

---

## 🧠 How it works

1. Add a company  
2. Add contacts  
3. Log your first interaction  
4. Schedule a follow-up  
5. Track progress in the dashboard  
6. Manage deals until won or lost  

---

## 💡 Why this project exists

As a developer, I kept losing opportunities simply because I didn’t follow up properly.

I didn’t need a complex CRM — I needed something simple, fast and focused on execution.

So I built one.

---

## 🧩 Core concepts

- **Interactions**: every commercial action logged
- **Follow-ups**: concrete next steps, not loose notes
- **Companies**: context for every opportunity
- **Quotations**: deal lifecycle tracking
- **Dashboard**: your daily operating center

## 🛠 Tech Stack

- Tailwind CSS 4
- Vite 8
- Pest 4
- MySQL

---

## ⚡ Getting started

composer install
cp .env.example .env
php artisan key:generate
php artisan migrate:fresh --seed
npm install
npm run build
php artisan serve
