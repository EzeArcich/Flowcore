# Application Flow

## Prospecting flow

1. User detects a potential company.
2. User creates the company record.
3. User adds one or more contacts.
4. User sends first outreach.
5. User registers the interaction.
6. User schedules a follow-up.
7. User receives a reply or sends another follow-up.
8. User creates a quotation when commercial interest is confirmed.
9. User updates negotiation status.
10. User closes as won or lost.

## Daily usage flow

1. Open dashboard.
2. Check overdue follow-ups.
3. Check follow-ups due today.
4. Execute actions.
5. Register every new interaction.
6. Update pipeline status.
