# Development Workflow

## Goal
Build the CRM in increments, always keeping the application usable.

## Implementation order
1. Migrations
2. Models and relationships
3. Company CRUD
4. Contact CRUD
5. Interaction registration
6. Follow-up management
7. Quotations
8. Dashboard
9. Search, filters and refinements

## Rule
Do not build advanced automation before the manual workflow is solid.

## Daily dev loop
- define a small milestone
- implement
- test manually
- refine UI only after the flow works
